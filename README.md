## What is Webcomics?
Webcomics (also known as online comics or Internet comics) are comics published on a website or mobile app. While many are published exclusively on the web, others are also published in magazines, newspapers or in comic books. (https://en.wikipedia.org/wiki/Webcomic)

## SMOKETOON
Smoketoon is an aplication for read webcomics online/offline for free, inspired by the line webtoon.

## Feature
* Read Webcomics online

* Search Webcomics
* Add Webcomic
* User Profile
* Make Your Own Webcomic

## Tech Stack
* React Native for the Mobile Frontend

## Prerequisites
* Make sure You had been install NodeJs in your system (https://nodejs.org/)
* Then install React Native (https://facebook.github.io/react-native/)

## Installation & Configuration
Follow these step to install

**Frontend:**
```console
$ git clone https://github.com/DumbWaysStudent/DW12CCJG9_webtoon.git
$ cd DW12CCJG9_webtoon
$ yarn install # for yarn
$ npm install # for npm
$ react-native run-ios #for ios
$ react-native run-android #for android
```
