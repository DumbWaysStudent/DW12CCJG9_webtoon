## Feature
* Read Webtoon Online
* Read Webtoon Offline (Coming Soon)
* Search Webtoon
* Add Favourite Webtoon
* User Profile
* Make Your Own Webtoon

## Tech Stack
* React Native for the Mobile Frontend

## Prerequisites
* Make sure You had been install NodeJs in your system (https://nodejs.org/)
* Then install React Native (https://facebook.github.io/react-native/)

## Installation & Configuration
Follow these step to install

**Frontend:**
```console
$ git clone https://github.com/DumbWaysStudent/DW12CCJG9_webtoon.git
$ cd DW12CCJG9_webtoon
$ yarn install #this project package manager is yarn not npm
$ react-native run-ios #for ios
$ react-native run-android #for android
```
